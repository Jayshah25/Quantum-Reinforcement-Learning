Getting Started
===============

Install qrl-qai:

.. code-block:: bash

   pip install qrl-qai

Or Install qrl-qai with PyTorch:

.. code-block:: bash

   pip install qrl-qai[torch]

Or from GitHub (latest):

.. code-block:: bash

   pip install git+https://github.com/Jayshah25/Quantum-Reinforcement-Learning.git
